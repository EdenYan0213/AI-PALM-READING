## 三、P0 详细执行方案

### 3.1 分享卡片设计 ✅ （本次已完成）

详见 `ShareCard.tsx` 组件，核心设计要点：
- **赛博玄学视觉**：深空蓝黑底 + 霓虹紫/青/粉渐变 + 发光网格
- **三大信息层**：手型身份 → 主线解读 → 稀有印记 → CTA
- **分享钩子**：二维码 + "扫码测你的专属手相"
- **社交属性**：人格标签、CP 匹配暗示、稀有度提示

### 3.2 CP 合拍深度开发

**现状问题**：CP 功能停留在概念阶段，缺少独立页面和传播机制。

**优化方案**：
1. **独立 CP 页面**：双人手型对比 + 雷达图匹配度 + 组合命名
2. **AI 匹配宣言**：生成一句专属文案如「逻辑脑 × 艺术魂，AI 判定 92% 天选搭档」
3. **分享卡片**：两人头像 + 匹配度 + 宣言，可一键分享
4. **陌生人匹配**：上传手相，AI 告诉你「今天街头会遇到什么手型的人」
5. **CP 周运势**：每周生成 CP 专属运势

**技术路径**：
- 新建 `CpPage.tsx` + `CpResultPage.tsx`
- 复用 `ShareCard` 组件，增加双人模式
- 后端 `POST /api/v1/cp/analyze` 接口已就绪

### 3.3 日签/周运势留存模块

**产品形态**：
- **每日手相运势**：用户每天拍一次（或一键生成），获得今日运势卡片
- **运势内容结构**：今日关键词 + 幸运色 + 幸运数字 + 一句话运势 + 宜/忌
- **周运势总结**：每周日推送上周围绕 + 下周展望

**数据驱动**：
- 基于日期 + 用户手型 + 随机种子生成，保证同日同手型结果一致
- 引入真实星象/节气数据增加可信度

### 3.4 内容投放策略

| 平台 | 内容形式 | KPI |
|---|---|---|
| **小红书** | 「AI 手相测评」笔记 + 结果截图 | 单篇 5000+ 阅读 |
| **抖音** | 「00 后 AI 算命」反应视频 | 单条 10w+ 播放 |
| **B 站** | 「赛博玄学科普」系列 | 品牌沉淀 |
| **微信朋友圈** | 分享卡片病毒传播 | 自然增长 |

**执行步骤**：
1. 找 50 个 KOC（粉丝 1000-5000）发测评笔记
2. 提供「专属折扣码」追踪转化
3. 引导用户发布 UGC 内容

## 四、P1 详细执行方案

### 4.1 私域转化漏斗

```
免费报告 → 稀有印记提示 → 点击查询 → 
跳转企业微信 → 专属解读师 1v1 → 付费深度服务（9.9元）
```

**关键优化点**：
- 稀有印记触发率控制在 15-25%，维持稀缺感
- 查询前加「正在分析...」3 秒等待，增加仪式感
- 企微欢迎语自动发送手相报告摘要
- 提供「分享给 3 个好友解锁免费深度解读」裂变机制

### 4.2 排行榜 + 好友系统

- **稀有手相排行榜**：谁的手相最稀有（凤凰眼、M 纹等）
- **CP 匹配度排行**：你和微信好友的匹配度
- **地区手相热力图**：「北京偏火型手，上海偏水型手」

## 五、P2 技术优化

### 5.1 关键点检测升级

**当前**：规则 + 模板 + LLM 拼装，不是真实图像识别。

**目标**：接入 MediaPipe Hands 做真实手掌关键点检测，提升可信度。

**实施路径**：
1. 前端集成 `@mediapipe/hands` 做实时关键点检测
2. 将检测结果（21 个关键点坐标）发送后端
3. 后端基于关键点计算手型、推断掌纹走向

### 5.2 用户体系

- 微信授权登录
- 用户手相档案（历史报告）
- 手相变化追踪（「本月你的生命线更清晰了」）

### 5.3 质量保障

- Jest + React Testing Library 前端测试
- JUnit 后端接口回归测试
- Playwright 端到端测试
- GitHub Actions CI 自动构建
