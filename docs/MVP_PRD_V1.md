# 手相研究所 MVP 执行版 PRD (V1.0)

## 1. 产品定义
- 产品名称: 手相研究所
- 产品定位: 结合 AI 与赛博玄学的娱乐手相解读产品
- 用户群体: 对星座、MBTI、轻玄学、分享玩法感兴趣的年轻用户
- Slogan: 你的互联网手相搭子，科学聊玄学。

## 2. MVP 范围
- 核心功能: 基础手相解读 + CP 合拍
- 技术策略: 大模型文案拼装 + 关键点检测流程演示
- 变现策略: 激励视频广告解锁深度内容
- 转化策略: 稀有印记查询引导企业微信

## 3. 信息架构
- 首页 (`light_1/code.html`)
- 拍照/上传 (`light_2/code.html`)
- AI 分析中 (`ai..._light/code.html`)
- 单人报告 (`light_3/code.html`)
- CP 合拍报告 (`cp_light/code.html`)
- 演示总入口 (`index.html`)

## 4. 用户流程
### 4.1 单人流程
1. 用户进入首页，点击开始 AI 识手相。
2. 进入拍照页，对齐掌型后拍照或上传。
3. 进入分析页，展示 3-5 秒动态文案和进度。
4. 进入报告页，先看免费总览。
5. 用户可选择看广告解锁深度报告。
6. 用户点击稀有印记查询，进入私域引导。

### 4.2 CP 合拍流程
1. 首页或报告页进入 CP 模式。
2. 查看匹配度、人格组合名与相处建议。
3. 用户可分享宿命卡片。
4. 用户可看广告解锁完整 CP 报告。

## 5. 核心能力拆解
### 5.1 AI 分析结构
- 观掌型: 土/风/火/水型手
- 看主线: 生命线、智慧线、感情线标签组合
- 找印记: 稀有标记概率触发 (如凤凰眼)

### 5.2 文案引擎
- 输入: 掌型 + 主线标签 + 印记 + 人格梗映射
- 输出风格: 赛博玄学、轻幽默、可社交传播
- 示例映射:
  - 水型手 -> 高敏感、直觉系、INFP 气质
  - 生命线锁链状 -> 体能波动像 Wi-Fi 信号

### 5.3 CP 引擎
- 匹配度: 70%-98% 区间
- 组合名: 例如 逻辑脑 x 艺术魂
- 文案: 互补点 + 摩擦点 + 相处 Tips

## 6. 页面与交互要求
### 6.1 首页
- 双入口明确: 开始识手相 / CP 合拍
- 展示品牌与核心卖点

### 6.2 拍照页
- 提示文案: 光线充足、男左女右
- 引导轮廓线 + 拍照按钮 + 相册入口

### 6.3 分析页
- 进度与状态文案轮播:
  - 正在扫描掌型编码...
  - 正在破译感情线波段...
  - 正在捕获稀有印记...
- 完成后自动跳转报告

### 6.4 报告页
- 免费层: 掌型 + 三主线简述
- 付费层: 广告解锁后显示深度内容
- 私域层: 稀有印记查询按钮与引导弹层

### 6.5 CP 页
- 显示匹配度、组合、主维度对比
- 可分享卡片
- 广告解锁完整 CP 报告

## 7. 关键指标
- 分享率: 报告分享 / CP 卡片分享
- 广告解锁率: 点击率 + 完播率
- 私域转化率: 印记入口到加微
- CP 发起率: CP 入口点击到报告生成

## 8. 验收标准 (V1.0)
- 可从首页完整走通单人流程。
- 分析页 3-5 秒内自动完成并跳转。
- 单人报告至少展示 1 个免费总览区块。
- 广告解锁后能展示深度内容区域。
- 稀有印记入口可触发私域引导弹层。
- CP 页面具备匹配展示、分享、解锁能力。

## 9. V1.1+ 迭代方向
- 校对模式: 用户手动微调识别点位
- 报告皮肤: 主题切换、卡面模板
- 日签/周运势: 留存模块
- 好友排行: 社交裂变玩法
